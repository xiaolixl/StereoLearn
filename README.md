# StereoLearn

**An Online Platform for Learning and Automatic Grading of Organic Structure Representations**

🌐 **Website**: [https://en.stereolearn.chemcognitive.online/](https://en.stereolearn.chemcognitive.online/)

StereoLearn is a web-based platform designed to help students develop representational competency in organic chemistry through interactive e-learning modules and a robust automated grading system. The platform supports scaffolded practice and high-stakes assessment of dash–wedge drawings, Fischer projections, and Haworth projections.
